KiduNotifier is the easiest marketing platform that leverages and automates the real power of website notifications to instantly increase trust, conversions and sales. 20+ notification styles to increase user attention to your relevant products.

== Installation ==
1. Download and unzip the plugin. Upload the unzipped folder to the wp-contents/plugins folder of your WordPress installation directory.
2. Active the plugin from the WordPress Plugin administration page.
3. Copy paste your **KiduNotifier Pixel API Key** under settings page **(Settings > KiduNotifier)**
